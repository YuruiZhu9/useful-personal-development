# Useful Personal Development

一个用于整理个人常用小工具、样式文件和开发辅助资源的仓库。

这些内容以"轻量、可复用、开箱即用"为主，方便在不同项目和开发环境中直接使用。

## Current Contents

- `pycharm-markdown.css`：优化 PyCharm 中 Markdown 文档的阅读样式
- `md转html的slides/`：Markdown 转 HTML 幻灯片相关资源
  - `md2slide/`：md2slide 技能定义文件，用于将 Markdown 内容转换为可交互的 HTML 幻灯片
  - `ai-coding-share-slides.html`：AI Coding 工具分享的 HTML 幻灯片示例
  - `AI coding工具分享.md`：AI Coding 工具分享的 Markdown 源文件
  - `Mastering_Agentic_Development.pptx`：Agentic 开发主题的演示文稿

## Usage

按需直接复制文件使用，或克隆仓库后在本地统一管理。

## Notes

仓库内容会持续补充，逐步沉淀更多实用的小工具与配置文件。
